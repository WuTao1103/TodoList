# 全栈Todo List应用

一个使用React和Spring Boot构建的现代化待办事项管理应用。

## 技术栈

### 前端
- React 18
- Material-UI
- Tailwind CSS
- Framer Motion (动画效果)
- Docker

### 后端
- Spring Boot 2.7
- Spring Data JPA
- MySQL 8
- Docker
- Lombok

## 功能特点

- ✨ 待办事项的增删改查
- 🎯 任务完成状态切换
- 💫 流畅的动画效果
- 🌓 支持深色/浅色主题
- 🔒 错误处理和日志记录
- 🚀 Docker容器化部署
- 🌐 跨域请求支持
